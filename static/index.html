Documentation for the Book Manager API

Path Details Request-type
/books Returns a list of books in the database in JSON format GET
/book/{{id}} Returns a detail view of the specified book id. Nest author details in JSON format GET
/authors Returns a list of authors in the database in JSON format GET
/author/{{id}} Returns a detail view of the specified author id GET
/author Creates a new author with the specified details - Expects a JSON body POST
/book Creates a new book with the specified details - Expects a JSON body POST
